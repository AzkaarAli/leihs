Class Contract
In: app/models/contract.rb
Parent: Document

A Contract is a Document containing ContractLine s. It gets created from an Order, once the Order is acknowledged by an InventoryPool manager.

The page "Flow" inside the models.graffle document shows the various steps though which a Document goes from Order to finally closed Contract.
