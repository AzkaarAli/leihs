Model

A Model is a type of a thing which is available inside an #InventoryPool for borrowing. If a customer wants to borrow a thing, he opens an #Order and chooses the appropriate Model. The #InventoryPool manager then hands him over an instance - an #Item - of that Model, in case one is still available for borrowing.

The description of the #Item class contains an example.

def package_models
  # NOTE assuming all roots have the same children structure
  items.each do |item|
    return item.children.collect(&:model) unless item.children.empty?
  end if is_package?
  return []
end
