Class NoticedError
In: vendor/plugins/newrelic_rpm/lib/newrelic/noticed_error.rb
Parent: Object

this class encapsulates an error that was noticed by RPM in a managed app. NOTE we do not capture the stack trace because RPM does not have access to the app‘s source code, so it has little value and is sizeable. We can add this later if customers demand it. (The stack trace will obviously be available elsewhere like the logs…)
