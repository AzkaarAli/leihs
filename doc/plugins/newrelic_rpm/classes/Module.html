add_method_tracer(method_name, metric_name_code, options = {})

Add a method tracer to the specified method. metric_name_code is ruby code that determines the name of the metric to be collected during tracing. As such, the code should be provided in ‘single quote’ strings rather than "double quote" strings, so that #{} evaluation happens at traced method execution time. Example: tracing a method :foo, where the metric name is the first argument converted to a string 

    add_method_tracer :foo, '#{args.first.to_s}'

statically defined metric names can be specified as regular strings push_scope specifies whether this method tracer should push the metric name onto the scope stack.

remove_method_tracer(method_name, metric_name_code)

Not recommended for production use, because tracers must be removed in reverse-order from when they were added, or else other tracers that were added to the same method may get removed as well.
